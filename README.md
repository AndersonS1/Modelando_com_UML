# 📱 Projeto iPhone Component - DIO

Este projeto tem como objetivo modelar e implementar, em Java, o componente iPhone , que reúne três funcionalidades principais:

- **Reprodutor Musical**
- **Aparelho Telefônico**
- **Navegador de Internet**
